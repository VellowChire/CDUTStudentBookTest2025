## 使用方式
  在[release](https://github.com/VellowChire/CDUTStudentBookTest2025/release)界面中下载最新版本docx文档即可。  
  ### 使用方法 1
    使用word打开文档，按下快捷键ctrl + F在侧边栏中搜索相关题目关键词寻找答案。
  ### 使用方法 2
    将文档喂给ai，复制题目向ai提问即可。
